# Smart-Students-Clearance-System
The Smart Students Clearance System is a web-based application that automates the student clearance process in tertiary institutions. It replaces manual procedures with a secure, role-based platform that allows students to request clearance online while institutional units review and approve requirements digitally.

Features

Role-based authentication (Admin, Staff, Student)

Online clearance request and approval workflow

Department, Faculty, Library, Bursary, and Registry modules

Real-time clearance status tracking

Secure data storage and access control
User Roles

Administrator

Department Staff

Faculty Staff

Library Staff

Bursary Staff

Registry Staff

Student
